Uses

Here is a list of the technology I use on a daily basis. To see other nifty things that people use, go check out /uses.

Hardware

  • Apple MacBook Pro M2 for work
  • Lenovo ThinkPad X1 Extreme for personal use
  • Mode Designs Envoy for a pleasant typing experience
  • Apple Magic Trackpad for moving the cursor on Apple computers
  • Logitech BRIO for remote meetings
  • Jarvis Bamboo Standing Desk for a pleasant space to work
  • Elgato Stream Deck for fancy physical hotkeys

Web Browsing

  • Firefox for a Google-free, fast browser
  • uBlock Origin for blocking ads and trackers
  • letsblock.it for generating uBlock Origin blocklists
  • Vimium for vim-like browser navigation
  • NoScript for blocking unwanted JavaScript execution
  • Decentraleyes for blocking tracking through CDNs
  • Privacy Badger for blocking invisible tracking
  • Stylus for injecting custom CSS
  • Tampermonkey for injecting custom JavaScript
  • SponsorBlock for skipping sponsorships on YouTube
  • User-Agent Switcher and Manager for spoofing User-Agent headers

Services

  • Proton for a privacy focused web application suite
  • NextDNS for blocking ads and trackers at DNS
  • SimpleLogin for generating anonymous email aliases
  • Bitwarden for secure password storage
  • Raindrop for cross-browser bookmarks
  • YNAB for basic monthly finance management

Tools

  • Visual Studio Code for writing software
  • JetBrains Mono for a beautiful monospaced font with ligatures
  • Fira Code for another amazing monospaced font with ligatures
  • Catppuccin for a beautiful pastel color theme
  • Dracula for a widely available dark pastel theme
  • Alacritty for a cross-platform, highly-configurable, performant terminal
  • Obsidian for my notes and general life organization
  • Structured for planning my day

Terminal

  • fish for a fast and friendly shell
  • starship for a super intuitive shell prompt
  • neovim for text editing
  • tmux for multiplexing terminals
  • git for code version management
  • git-delta for beautiful code diffs in git
  • nnn for filesystem navigation
  • btop for monitoring resource usage (replaces top)
  • mise for managing runtime versions
  • fzf for fuzzy finding things
  • zoxide for quickly hopping between directories (replaces cd)
  • ripgrep for finding file contents (replaces grep)
  • bat for viewing files (replaces cat)
  • lsd for listing directories (replaces ls)
  • xh for interacting with HTTP requests (replaces curl)
  • jq for processing and querying JSON
  • just for executing project commands (replaces make)
  • atuin for magical shell history

Healthcare

  • Apple Health for organizing health information
  • Gentler Streak for tracking workout energy expenditure
  • Hevy for planning workouts and tracking improvements
  • Cronometer for tracking macro and micronutrients
  • Yuka for checking for harmful additives
  • Pillow for tracking sleep quality
  • Withings for health monitoring hardware

Photography

  • Sony A7III Camera for taking photos
  • Sigma 24-70mm F2.8 DGDN A Lens for high-quality exposure
  • Sigma 50mm F2 DGDN C Lens for quick snapshots
  • Peak Design Travel Tripod for stable shots
  • Peak Design Cuff for securing my camera my wrist
  • Peak Design Capture for securing my camera to a pack

Audio

  • Spotify for streaming music and podcasts
  • DROP+THX AAA 789 Amplifier for amplifying audio
  • DROP+Grace SDAC for converting digital audio to analogue
  • Focusrite Scarlet Solo 4th Generation for analogue microphone input
  • Beyer Dynamic DT 990 Pro Headphones for casual listening
  • Sennheiser HD 600 Headphones for critical listening
  • Audio Technica AT2020 Microphone for capturing my voice

Everyday Carry

  • Peak Design Everyday Backpack 20L for carrying my things
  • Ridge Wallet for carrying my cards and cash
  • Spyderco Sage 5 for cutting things
  • Traveler's Company Notebook for taking handwritten notes
  • Uni Jestream Edge 0.38mm Ballpoint Pen for writing things
  • Kobo Sage for reading books
